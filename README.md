# Project Praktikum PWEB 
This is project to complete final assignment of praktikum pemrograman web 

## Description
This project is focused on CRUD feature, this project is a website build like a book store, where the owner can add book, edit exsisting book and delete book.

## Getting Started
### Dependencies
* Node.js 18+
* React
* Axios
* React Router DOM
* Express
* MySQL
* Multer (upload gambar)
* CORS

### Installing
* **Clone this Repository**
```
git clone https://github.com/Alexandro-K/project-praktikum-pweb.git
```
* **Frontend**
```
cd frontend
npm install
npm install axios react-router-dom
```
* **Backend**
```
cd backend
npm install
npm install express mysql cors multer path
```
* **Database**
```
Download the database on /database
Import the database on mysql
```
### Executing Program
* **Frontend**
```
cd frontend
npm run dev
```
* **Backend**
```
cd backend
node index.js
```
